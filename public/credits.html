eSIM 管理工具是一款开源的 uTools 插件，旨在帮助用户轻松管理 eSIM 配置文件与设备。插件提供直观界面，可导入、导出并组织 eSIM 数据。

eSIM Manager is an open-source uTools plugin designed to help users manage eSIM profiles and devices. It offers an intuitive interface to import, export, and organize eSIM data.
